# Internationalisierung (i18n)

## Sprachen
- Deutsch (Standard)
- Englisch

## Labels überschreiben

```typescript
import { totpPlugin, defaultLabels, englishLabels } from '@democracy-deutschland/payload-totp-plugin';

totpPlugin({
  localization: {
    labels: englishLabels,
  },
});

// Einzelne Labels

totpPlugin({
  localization: {
    labels: {
      ...defaultLabels,
      loginTitle: 'Mein Login',
    },
  },
});
```
