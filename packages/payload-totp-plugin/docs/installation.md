# Installation

```bash
# pnpm (empfohlen)
pnpm add @democracy-deutschland/payload-totp-plugin

# npm
npm install @democracy-deutschland/payload-totp-plugin

# yarn
yarn add @democracy-deutschland/payload-totp-plugin
```

## Peer Dependencies

```bash
pnpm add otplib qrcode
pnpm add -D @types/qrcode
```
